### Vagrant
 I use Vagrant in this project.
 Synch with NFS dir -> ./
 
 ```
    $ vagrant up
 ```
  
 Add to your hosts file:
 
 ```
  192.168.33.100  series_reader.local
 ```

### Run

  Go to  [http://series_reader.local/](http://series_reader.local/)
